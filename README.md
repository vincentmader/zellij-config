# Zellij Configuration

For switching Zellij modes, we're using the following key mappings:

| Key Mapping | Mode    |
| ----------- | ------- |
| `Ctrl + g`  | Lock    |
| `Ctrl + p`  | Pane    |
| `Ctrl + t`  | Tab     |
| `Ctrl + n`  | Resize  |
| `Ctrl + f`  | Move    |
| `Ctrl + s`  | Search  |
| `Ctrl + o`  | Session |
| `Ctrl + x`  | Quit    |
